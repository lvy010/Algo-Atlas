### 好用的笔记工具

- **draw.io**
  电脑绘图工具
  写笔记的时候思考如何清晰的让别人也可以看懂 可以帮助更好的理解学习

- **xmind**
  导图工具
  以前几天每日一题中的区间贪心为例，我们可以借助导图来清晰地梳理题解。

- **Goodnotes**
  ipad 手绘工具，多种颜色画笔，可以清晰的对下面三个部分进行写写画画

  1.写代码前：模拟代码，例如 N 皇后、有效的数独。

  2.写代码后：梳理代码，以分析正则表达式为例，提炼关键状态转移并总结心得。

- **LICEcup**
  录屏 gif 动图，可用于演示递增的三元子序列、最长连续递增序列、买卖股票的最佳时机等题目的贪心过程。

#### 递归，搜索与回溯

搜索是递归中的一个分支

回溯是搜索中的一个分支

因此我们学习顺序是递归、搜索、回溯。

#### 动态规划

动态规划最核心的思想，就在于 拆分子问题，记住过往，减少重复计算。

0基础可以先通过第 N 个泰波那契数、三步问题等题目建立直觉。

> tips: 多刷题多体会~

#### 贪心算法

与其说是贪心算法，不如说是贪心策略。

贪心策略：解决问题的策略( 局部最优 —> 全局最优)。

把解决问题的过程分为若干步；

解决每一步的时候，都选择当前看起来 “最优的” 解法；

“希望” 得到全局最优解。

遇到不会的贪心题，很正常，把心态放平。

前期学习的时候，把重点放在贪心的策略上，把这个策略当成经验吸收。往后遇到相同类型的题目时可以用经验去解决这道问题。
当知道贪心是正确的时候，要想到如何去证明。

可以从柠檬水找零等题目入手，体会贪心策略如何落地。
